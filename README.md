# vagrant-lamp

Создание вирутальной машины с Linux и установка на неё XAMPP:
```
  vagrant up
```

Подключиться к машине по ssh:
```
  vagrant ssh
```
  
Выключить машину:
```
  vagrant halt
```


## Подключившись по ssh можем сделать следующее

Запуск XAMPP:
```
  sudo /opt/lampp/lampp start
```
  
Остановка XAMPP:
```
  sudo /opt/lampp/lampp stop
```
